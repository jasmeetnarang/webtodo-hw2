webdev-todoList
===============

###Usage

1. Clone Repo:
 
       git clone https://github.com/jasmeetnarang/webtodo-hw2.git

2. Browse to clone directory

        cd webdevtodo-hw2
    
3. Install NPM packages
        
        npm i
4. Start Node Server
 
        nodemon sever.js
4. Start React Server
 
        npm start
       
6. Browse to http://localhost:8080 or see live at https://todohw2.herokuapp.com/

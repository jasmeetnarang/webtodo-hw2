webdev-todoList
===============

###Usage

1. Clone Repo:
 
       git clone https://github.com/jasmeetnarang/webtodo-hw2.git

2. Browse to clone directory

        cd webdevtodo-hw2
    
3. Install NPM packages
        
        npm i
4. Start Node Server
 
        nodemon sever.js
       
5. Browse to http://localhost:8080 or see live at
